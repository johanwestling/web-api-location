# Web API - Location

A basic example of the Location Web API.

## Setup

1. Install Node.js (v10 or later)
1. Clone this repository:
    ```bash
    git clone git@github.com:johanwestling/web-api-location.git
    ```
1. Install dependencies:
    ```bash
    npm i
    ```
1. Run dev server:
    ```bash
    npm start
    ```
1. Build for production:
    ```bash
    npm run build
    ```
